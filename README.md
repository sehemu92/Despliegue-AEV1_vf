--Juego del ahorcado--
Cada jugador tendrá un total de 6 vidas.

El jugador debe de ir indicando las letras para acertar la palabra del juego,
una vez acierta gana, de lo contrario cada letra que no este contenida en la
palabra a adivinar restará 1 vida.

Las letras utilizadas y que no esten en la palabra a adivinar se indicaran en la
parte inferior del juego, para que el usuario no las repita.
